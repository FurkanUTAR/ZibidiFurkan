HTML Nedir?
HTML (Hyper Text Markup Language), Türkçesiyle "Zengin Metin İşaretleme Dili", web sayfalarını oluşturmak için kullanılan temel bir işaretleme dilidir. 1980'li yıllarda Tim Berners-Lee tarafından geliştirilen bu dil, 1990'larda web standardı haline gelmiştir. Berners-Lee, aynı zamanda web standartlarını belirleyen World Wide Web Consortium (W3C) organizasyonunun da kurucusudur. HTML'in ilk sürümlerinde, sayfalar yalnızca metinlerden oluşuyordu ve görsel olarak oldukça sınırlıydı. Etiketler büyük veya küçük harfle yazılabiliyordu. Daha sonraları XHTML (eXtensible HTML) geliştirildi ve HTML'e daha katı kurallar getirildi. Örneğin, tüm etiketlerin küçük harfle yazılması gerekiyordu ve belgelerin hiyerarşik bir yapıya sahip olması zorunlu hale geldi. 1996 yılında CSS (Cascading Style Sheets) ortaya çıktı ve HTML ile oluşturulan web sayfaları, artık daha estetik ve esnek bir yapıya kavuştu. HTML zaman içinde gelişerek daha fazla özellik kazandı. Günümüzde HTML, CSS ve JavaScript ile birlikte kullanılarak modern, dinamik ve kullanıcı dostu web siteleri oluşturulmasını sağlar. HTML’in asıl gücü, PHP, ASP, ASP.NET gibi web programlama dilleriyle ve JavaScript gibi betik dilleriyle uyum içinde çalışmasından gelir. Bu sayede sadece içerik düzenlemekle kalmaz, etkileşimli ve dinamik web sayfaları da oluşturulabilir.

Temel Kodlar
1. Temel HTML Sayfa Yapısı

Her HTML belgesi aşağıdaki temel yapıyla başlar:

    <!DOCTYPE html>
    <html lang="tr">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Benim İlk Sayfam</title>
    </head>
    <body>
        <h1>Merhaba Dünya!</h1>
        <p>Bu benim ilk HTML sayfam.</p>
    </body>
    </html>
2. En Çok Kullanılan HTML Etiketleri

Başlıklar (Headings)

                            <h1>Ana Başlık</h1>
                            <h2>Alt Başlık</h2>
                            <h3>Daha Küçük Başlık</h3>
                    Bu şekilde <h6>  </h6> 'ya kadar gidiyor.

Paragraf (Paragraph)

                            <p>Bu bir paragraftır.</p>

Listeler (Lists)

Sırasız Liste (Madde İşaretli):

                            <ul>
                                <li>Elma</li>
                                <li>Armut</li>
                                <li>Muz</li>
                            </ul>

Sıralı Liste (Numaralı):

                            <ol>
                                <li>Adım 1</li>
                                <li>Adım 2</li>
                                <li>Adım 3</li>
                            </ol>
3. Örnek Bir HTML Sayfası

İşte basit bir web sayfası örneği:

    <!DOCTYPE html>
    <html lang="tr">
    <head>
        <meta charset="UTF-8">
        <title>Kişisel Web Sayfam</title>
    </head>
    <body>
        <h1>Hoş Geldiniz!</h1>
        <p>Bu benim kişisel web sayfam. Aşağıda bazı bilgiler bulabilirsiniz.</p>

        <h2>Hobilerim</h2>
        <ul>
            <li>Yazılım Geliştirme</li>
            <li>Müzik Dinlemek</li>
            <li>Yürüyüş Yapmak</li>
        </ul>

        <h2>İletişim</h2>
            <label>E-posta:</label>
            <input type="email" placeholder="E-posta adresiniz"><br>
            <button>Gönder</button>
    </body>
    </html>
4. HTML Etiketleri

HTML'de kullanılan bazı temel etiketler:

  1. <img>: Resim ekler.
  2. <div>: Bölüm oluşturur.
  3. <span>: Satır içi bölüm oluşturur.
  4. <table>: Tablo oluşturur.
CSS Nedir?
CSS (Cascading Style Sheets), Türkçesiyle Basamaklı Stil Sayfaları, HTML ile oluşturulan web sayfalarının tasarımını ve görünümünü düzenlemek için kullanılan bir stil dilidir. CSS sayesinde yazı tipi, renk, arka plan, kenarlık, hizalama ve daha birçok görsel öğe kolayca yönetilebilir. CSS’ten önce, her HTML etiketi için ayrı ayrı stil tanımlaması yapmak gerekiyordu. Bu yöntem, büyük ve karmaşık projelerde yönetimi zorlaştırıyordu. CSS ile birlikte bu sorun ortadan kalktı; artık tek bir stil dosyası ile tüm sayfanın tasarımını düzenlemek mümkün hale geldi. CSS’in "cascading" (basamaklı) olarak adlandırılmasının sebebi, hiyerarşik bir yapı kullanarak stillerin belirli öğelere uygulanabilmesidir. Örneğin, bir div içerisindeki paragrafın ilk satırına şu şekilde erişilebilir: div > p:first-line { color: red; } Bu sayede belirlenen stil, tüm sayfalarda tek bir kod satırıyla uygulanabilir. CSS öğrenmesi ve kullanması oldukça kolaydır. Web geliştirme sürecinde HTML ve CSS, temel yapı taşlarıdır ve genellikle yazılım öğrenmeye başlamadan önce bu iki teknolojiye hakim olunması tavsiye edilir. CSS’i öğrendikçe, daha önce yapamadığınız birçok tasarımı çok daha kolay bir şekilde gerçekleştirebildiğinizi fark edeceksiniz.
JavaScript Nedir?
JavaScript, web sayfalarına dinamiklik ve etkileşim kazandırmak için kullanılan bir programlama dilidir. HTML sayfalarının yalnızca metin ve görsellerden oluşan statik yapısını değiştirerek, kullanıcı ile etkileşime girebilen sayfalar oluşturmayı sağlar. JavaScript sayesinde bir butona tıklandığında sayfanın içeriğini değiştirebilir, formları doğrulayabilir, animasyonlar ekleyebilir veya bir kullanıcının hareketlerini takip edebilirsiniz. Başlangıçta yalnızca web tarayıcıları için geliştirilen JavaScript, günümüzde Node.js gibi teknolojiler sayesinde sunucu tarafında da çalışabilir hale gelmiştir. Bu, JavaScript’in artık yalnızca tarayıcılarda değil, masaüstü ve mobil uygulamalarda da kullanılmasını sağlamaktadır. JavaScript'in en güçlü yanlarından biri, HTML ve CSS ile birlikte çalışabilmesi ve web sayfalarına interaktif öğeler ekleyebilmesidir. Örneğin, aşağıdaki JavaScript kodu bir butona tıklandığında ekrana mesaj verir: document.getElementById("myButton").addEventListener("click", function() { alert("Butona tıklandı!"); }); JavaScript öğrenmesi ve kullanması oldukça kolaydır. Temel HTML ve CSS bilgisi olan herkes, JavaScript ile daha gelişmiş ve kullanıcı dostu web siteleri oluşturabilir.
