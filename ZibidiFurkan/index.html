HTML Nedir?
HTML (Hyper Text Markup Language), Türkçesiyle "Zengin Metin İşaretleme Dili", web sayfalarını oluşturmak için kullanılan temel bir işaretleme dilidir. 1980'li yıllarda Tim Berners-Lee tarafından geliştirilen bu dil, 1990'larda web standardı haline gelmiştir. Berners-Lee, aynı zamanda web standartlarını belirleyen World Wide Web Consortium (W3C) organizasyonunun da kurucusudur. HTML'in ilk sürümlerinde, sayfalar yalnızca metinlerden oluşuyordu ve görsel olarak oldukça sınırlıydı. Etiketler büyük veya küçük harfle yazılabiliyordu. Daha sonraları XHTML (eXtensible HTML) geliştirildi ve HTML'e daha katı kurallar getirildi. Örneğin, tüm etiketlerin küçük harfle yazılması gerekiyordu ve belgelerin hiyerarşik bir yapıya sahip olması zorunlu hale geldi. 1996 yılında CSS (Cascading Style Sheets) ortaya çıktı ve HTML ile oluşturulan web sayfaları, artık daha estetik ve esnek bir yapıya kavuştu. HTML zaman içinde gelişerek daha fazla özellik kazandı. Günümüzde HTML, CSS ve JavaScript ile birlikte kullanılarak modern, dinamik ve kullanıcı dostu web siteleri oluşturulmasını sağlar. HTML’in asıl gücü, PHP, ASP, ASP.NET gibi web programlama dilleriyle ve JavaScript gibi betik dilleriyle uyum içinde çalışmasından gelir. Bu sayede sadece içerik düzenlemekle kalmaz, etkileşimli ve dinamik web sayfaları da oluşturulabilir.

Temel HTML Kodları
1. Temel HTML Sayfa Yapısı

Her HTML belgesi aşağıdaki temel yapıyla başlar:

    <!DOCTYPE html>
    <html lang="tr">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Benim İlk Sayfam</title>
    </head>
    <body>
        <h1>Merhaba Dünya!</h1>
        <p>Bu benim ilk HTML sayfam.</p>
    </body>
    </html>
2. En Çok Kullanılan HTML Etiketleri

Başlıklar (Headings)

                            <h1>Ana Başlık</h1>
                            <h2>Alt Başlık</h2>
                            <h3>Daha Küçük Başlık</h3>
                            Bu şekilde <h6></h6> 'ya kadar gidiyor.

Paragraf (Paragraph)

                            <p>Bu bir paragraftır.</p>

Listeler (Lists)

Sırasız Liste (Madde İşaretli):

                            <ul>
                                <li>Elma</li>
                                <li>Armut</li>
                                <li>Muz</li>
                            </ul>

Sıralı Liste (Numaralı):

                            <ol>
                                <li>Adım 1</li>
                                <li>Adım 2</li>
                                <li>Adım 3</li>
                            </ol>

Farklı Sıralama Tipleri

<ol> etiketinde type özelliğiyle numaralandırma stilini değiştirebilirsin:

Harflerle (Küçük harfler):
                            <ol type="a">
                                <li>Birinci Madde</li>
                                <li>İkinci Madde</li>
                                <li>Üçüncü Madde</li>
                            </ol>
Büyük Harflerle:
                            <ol type="A">
                                <li>A maddesi</li>
                                <li>B maddesi</li>
                            </ol>
Roma Rakamları (Küçük):
                            <ol type="i">
                                <li>Birinci</li>
                                <li>İkinci</li>
                            </ol>
Roma Rakamları (Büyük):
                            <ol type="I">
                                <li>I. Madde</li>
                                <li>II. Madde</li>
                            </ol>
Başlangıç Numarasını Değiştirme

Listelerin hangi sayıyla başlayacağını belirlemek için start özelliği kullanılır:

                            <ol start="5">
                                <li>Beşinci</li>
                                <li>Altıncı</li>
                            </ol>
3. Örnek Bir HTML Sayfası

İşte basit bir web sayfası örneği:

    <!DOCTYPE html>
    <html lang="tr">
    <head>
        <meta charset="UTF-8">
        <title>Kişisel Web Sayfam</title>
    </head>
    <body>
        <h1>Hoş Geldiniz!</h1>
        <p>Bu benim kişisel web sayfam. Aşağıda bazı bilgiler bulabilirsiniz.</p>

        <h2>Hobilerim</h2>
        <ul>
            <li>Yazılım Geliştirme</li>
            <li>Müzik Dinlemek</li>
            <li>Yürüyüş Yapmak</li>
        </ul>

        <h2>İletişim</h2>
            <label>E-posta:</label>
            <input type="email" placeholder="E-posta adresiniz"><br>
            <button>Gönder</button>
    </body>
    </html>
4. HTML Etiketleri

HTML'de kullanılan bazı temel etiketler:

  1. <img>: Resim ekler.
  2. <div>: Bölüm oluşturur.
  3. <span>: Satır içi bölüm oluşturur.
  4. <table>: Tablo oluşturur.
CSS Nedir?
CSS (Cascading Style Sheets), Türkçesiyle Basamaklı Stil Sayfaları, HTML ile oluşturulan web sayfalarının tasarımını ve görünümünü düzenlemek için kullanılan bir stil dilidir. CSS sayesinde yazı tipi, renk, arka plan, kenarlık, hizalama ve daha birçok görsel öğe kolayca yönetilebilir. CSS’ten önce, her HTML etiketi için ayrı ayrı stil tanımlaması yapmak gerekiyordu. Bu yöntem, büyük ve karmaşık projelerde yönetimi zorlaştırıyordu. CSS ile birlikte bu sorun ortadan kalktı; artık tek bir stil dosyası ile tüm sayfanın tasarımını düzenlemek mümkün hale geldi. CSS’in "cascading" (basamaklı) olarak adlandırılmasının sebebi, hiyerarşik bir yapı kullanarak stillerin belirli öğelere uygulanabilmesidir. Örneğin, bir div içerisindeki paragrafın ilk satırına şu şekilde erişilebilir: div > p:first-line { color: red; } Bu sayede belirlenen stil, tüm sayfalarda tek bir kod satırıyla uygulanabilir. CSS öğrenmesi ve kullanması oldukça kolaydır. Web geliştirme sürecinde HTML ve CSS, temel yapı taşlarıdır ve genellikle yazılım öğrenmeye başlamadan önce bu iki teknolojiye hakim olunması tavsiye edilir. CSS’i öğrendikçe, daha önce yapamadığınız birçok tasarımı çok daha kolay bir şekilde gerçekleştirebildiğinizi fark edeceksiniz.

Temel CSS Kodları
1. CSS Ekleme Yöntemleri

1. Satır İçi CSS (Inline)

        <p style="color: blue;">Bu mavi renkte bir paragraftır.</p>

2. Dahili CSS (Internal)

        <head>
            <style>
                body {
                    background-color: lightgray;
                }
            </style>
        </head>

3. Harici CSS (External)

        <link rel="stylesheet" href="stil.css">
2. Sık Kullanılan CSS Özellikleri
  1. color: Yazı rengini ayarlar.
  2. background-color: Arka plan rengini belirler.
  3. font-size: Yazı boyutunu belirler.
  4. text-align: Metni hizalar (left, center, right).
  5. margin: Dış boşlukları ayarlar.
  6. padding: İç boşlukları ayarlar.
  7. border: Kenarlık ekler.
3. Sınıf (class) ve Kimlik (id) Kullanımı

Belirli elementlere özel stil vermek için class ve id kullanılır.

        <style>
            .kirmizi {
                color: red;
            }

            #ozel {
                font-weight: bold;
            }
        </style>

        <p class="kirmizi">Bu kırmızı yazıdır.</p>
        <p id="ozel">Bu kalın yazıdır.</p>
4. CSS Box Model (Kutu Modeli)

Her HTML öğesi bir kutu gibi düşünülür. Kutu modeli şu alanlardan oluşur:

  • Content: İçerik
  • Padding: İçerik ile kenarlık arası boşluk
  • Border: Kenarlık
  • Margin: Dış boşluk
        <style>
            div {
                width: 200px;
                padding: 20px;
                border: 2px solid black;
                margin: 30px;
            }
        </style>
5. Geçiş Efektleri (CSS Transitions)

Geçiş efektleri, bir CSS özelliği değiştiğinde bu değişimin yumuşak bir şekilde gerçekleşmesini sağlar. Örneğin, bir kutunun rengi ya da boyutu yavaşça değişebilir.

        <style>
            .gecis {
                width: 100px;
                height: 100px;
                background-color: blue;
                transition: all 0.5s ease;
            }

            .gecis:hover {
                background-color: red;
                width: 150px;
            }
        </style>

        <div class="gecis"></div>

transition özelliğinde şunlar ayarlanabilir:

  • property: Hangi özelliğin geçişle değişeceği (örneğin background-color).
  • duration: Geçiş süresi (örneğin 0.5s).
  • timing-function: Geçişin hız eğrisi (örneğin ease, linear, ease-in).
  • delay: Geçişin başlaması için bekleme süresi (isteğe bağlı).
6. Dönüşümler (CSS Transform)

transform özelliğiyle bir öğeyi döndürebilir, ölçeklendirebilir, kaydırabilir veya eğebilirsiniz. Genellikle hover gibi etkileşimlerle birlikte kullanılır.

Örnekler:

        <style>
            .donusum {
                width: 100px;
                height: 100px;
                background-color: green;
                transition: transform 0.3s;
            }

            .donusum:hover {
                transform: rotate(45deg); /* 45 derece döndürme */
            }

            .buyut:hover {
                transform: scale(1.2); /* %120 büyütme */
            }

            .kaydir:hover {
                transform: translateX(50px); /* Sağa kaydırma */
            }
        </style>

        <div class="donusum"></div>
        <div class="donusum buyut"></div>
        <div class="donusum kaydir"></div>

Yaygın kullanılan transform değerleri:

  • rotate(degree): Döndürür.
  • scale(x, y): Büyütür/küçültür.
  • translate(x, y): Yer değiştirir (x: yatay, y: dikey).
  • skew(x, y): Eğme efekti verir.
JavaScript Nedir?
JavaScript, web sayfalarına dinamiklik ve etkileşim kazandırmak için kullanılan bir programlama dilidir. HTML sayfalarının yalnızca metin ve görsellerden oluşan statik yapısını değiştirerek, kullanıcı ile etkileşime girebilen sayfalar oluşturmayı sağlar. JavaScript sayesinde bir butona tıklandığında sayfanın içeriğini değiştirebilir, formları doğrulayabilir, animasyonlar ekleyebilir veya bir kullanıcının hareketlerini takip edebilirsiniz. Başlangıçta yalnızca web tarayıcıları için geliştirilen JavaScript, günümüzde Node.js gibi teknolojiler sayesinde sunucu tarafında da çalışabilir hale gelmiştir. Bu, JavaScript’in artık yalnızca tarayıcılarda değil, masaüstü ve mobil uygulamalarda da kullanılmasını sağlamaktadır. JavaScript'in en güçlü yanlarından biri, HTML ve CSS ile birlikte çalışabilmesi ve web sayfalarına interaktif öğeler ekleyebilmesidir. Örneğin, aşağıdaki JavaScript kodu bir butona tıklandığında ekrana mesaj verir: document.getElementById("myButton").addEventListener("click", function() { alert("Butona tıklandı!"); }); JavaScript öğrenmesi ve kullanması oldukça kolaydır. Temel HTML ve CSS bilgisi olan herkes, JavaScript ile daha gelişmiş ve kullanıcı dostu web siteleri oluşturabilir.

Temel JavaScript Kodları
1. JavaScript Ekleme Yöntemleri

1. Satır İçi JavaScript

            <button onclick="alert('Butona tıkladın!')">Tıkla</button>

2. Dahili JavaScript

                    <script>
                        console.log("Konsola yazı yazıldı.");
                    </script>

3. Harici JavaScript

                    <script src="script.js"></script>
2. Değişkenler (Variables)

Veri saklamak için kullanılır:

                    let isim = "Ali";
                    const yas = 17;
                    var sehir = "İstanbul";
3. Koşullu İfadeler (if-else)

Şartlara göre farklı işlemler yapılmasını sağlar:

                    let not = 85;

                    if (not >= 50) {
                        alert("Geçtin!");
                    } else if (not == 50) {
                        alert("Sınırdasın :p");
                    } else {
                        alert("Kaldın.");
                    }
4. Fonksiyonlar (Functions)

Belirli görevleri yapmak için kullanılır:

                    function selamla() {
                        alert("Selam!");
                    }

                    selamla();
5. Olaylar (Events)

Kullanıcının sayfa ile etkileşimini yakalar:

    <button onclick="renkDegistir()">Rengi Değiştir</button>

    <script>
        function renkDegistir() {
            document.body.style.backgroundColor = "lightblue";
        }
    </script>
6. Elementlere Erişim (getElementById)

HTML elementlerine ID kullanarak erişebilirsin:

    <p id="mesaj">Merhaba!</p>
    <button onclick="degistir()">Yazıyı Değiştir</button>

    <script>
        function degistir() {
            document.getElementById("mesaj").innerText = "Butona tıkladın!";
        }
    </script>
7. addEventListener Kullanımı

Olayları daha profesyonel şekilde tanımlar:

    <button id="btn">Bana Bas</button>

    <script>
        document.getElementById("btn").addEventListener("click", function() {
            alert("Butona bastın!");
        });
    </script>
8. Diziler (Arrays)

Birden fazla veri tutmak için kullanılır:

                    let meyveler = ["Elma", "Muz", "Portakal"];
                    alert(meyveler[0]); // Elma
9. Döngüler (Loops)

Bir işlemi tekrar tekrar yapmak için:

                    let sayilar = [1, 2, 3, 4, 5];

                    for (let i = 0; i < sayilar.length; i++) {
                        console.log(sayilar[i]);
                    }
10. Kullanıcıdan Veri Almak

prompt

                    let ad = prompt("Adınız nedir?");
                    alert("Merhaba " + ad);

confirm

                    let cevap = confirm("Silmek istediğine emin misin?");
                    if (cevap) {
                        alert("Silindi.");
                    } else {
                        alert("İptal edildi.");
                    }
11. Zamanlayıcılar (setTimeout ve setInterval)

setTimeout (Belirli süre sonra çalıştır)

                    setTimeout(function() {
                        alert("3 saniye sonra bu mesaj görünecek.");
                    }, 3000);

setInterval (Belirli aralıklarla tekrar et)

                    setInterval(function() {
                        console.log("Her 2 saniyede bir bu yazı çıkar.");
                    }, 2000);
12. HTML İçeriğini Değiştirmek
    <p id="icerik">Eski içerik</p>
    <button onclick="guncelle()">İçeriği Güncelle</button>

    <script>
        function guncelle() {
            document.getElementById("icerik").innerHTML = "<strong>Yeni içerik</strong>";
        }
    </script>
